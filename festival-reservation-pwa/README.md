# 학교 축제 부스 예약 PWA 시제품

## 들어 있는 기능
- 모바일 부스 목록/상세 화면
- 시간대별 남은 자리 표시
- 학번·이름 예약
- 같은 부스 및 같은 시간 중복 예약 방지
- 내 예약 확인/취소
- 운영자 예약 현황 및 CSV 저장
- PWA 홈 화면 설치
- 알림 권한 및 테스트 알림
- 오프라인 기본 화면 캐시

## 시제품 운영자 비밀번호
`2026`

## 실행 방법
PWA 기능은 `file://`로 직접 열면 제한됩니다. 로컬 서버 또는 HTTPS 호스팅에서 실행하세요.

### 간단한 로컬 실행
```bash
python -m http.server 8080
```
그 뒤 브라우저에서 `http://localhost:8080`으로 접속합니다.

### GitHub Pages 배포
1. 이 폴더의 파일을 GitHub 저장소 최상위에 업로드합니다.
2. 저장소 Settings → Pages에서 배포 브랜치를 선택합니다.
3. 생성된 HTTPS 주소를 QR코드로 만들면 됩니다.

## 중요한 제한
현재 예약 정보는 각 휴대폰의 `localStorage`에 저장됩니다. 따라서 다른 학생의 예약과 실시간으로 공유되지 않습니다.
실제 축제 운영용으로 사용하려면 Firebase Authentication/Firestore/Cloud Functions/FCM을 연결해야 합니다.
